рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бһӮбһҺбһ“бһ¶бһӣбһёбһҳбһёбһҸ $\lim_{x \to 5} (2x - 3)$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҮбҹҶбһ“бһҪбһҹ $x = 5$ бһ…бһјбһӣбҹ– $\lim_{x \to 5} (2x - 3) = 2(5) - 3 = 10 - 3 = 7$бҹ”
бһӮбһҺбһ“бһ¶бһӣбһёбһҳбһёбһҸ $\lim_{x \to +\infty} \dfrac{20x^2 - 7}{4x^2 + 9}$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҡбһ¶бһ„бһҳбһ·бһ“бһҖбҹҶбһҺбһҸбҹӢ $\dfrac{\infty}{\infty}$бҹ– бһҷбһҖбһҸбһҪбһҠбһәбһҖбҹ’бһҡбҹҒбһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбһҳбһҖбһӮбһҺбһ“бһ¶ бһӮбһә $\lim_{x \to +\infty} \dfrac{20x^2}{4x^2} = \dfrac{20}{4} = 5$бҹ”
бһӮбһҺбһ“бһ¶бһӣбһёбһҳбһёбһҸ $\lim_{x \to 10} \dfrac{x^2 - 100}{x - 10}$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҡбһ¶бһ„бһҳбһ·бһ“бһҖбҹҶбһҺбһҸбҹӢ $\dfrac{0}{0}$бҹ– бһ”бҹҶбһ”бҹӮбһҖбһ—бһ¶бһӮбһҷбһҖ $(x-10)(x+10)$ бһҡбһҪбһ…бһҹбһҳбҹ’бһҡбһҪбһӣбһ“бһ№бһ„бһ—бһ¶бһӮбһ”бҹӮбһ„ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $\lim_{x \to 10} (x + 10) = 10 + 10 = 20$бҹ”
бһӮбһҺбһ“бһ¶бһӣбһёбһҳбһёбһҸ $\lim_{x \to 2} \dfrac{x^2 - 3x + 2}{x - 2}$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһ”бҹҶбһ”бҹӮбһҖбһ—бһ¶бһӮбһҷбһҖбҹ– $x^2 - 3x + 2 = (x-2)(x-1)$бҹ”
  • бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһҸбҹ’бһҸбһ¶бһҡбһҪбһҳ $(x-2)$ бһ…бҹҒбһү бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $\lim_{x \to 2} (x - 1) = 2 - 1 = 1$бҹ”
бһҡбһҖбһҠбҹҒбһҡбһёбһңбҹҒбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = x^4 - 4x^2 + 5x + 1$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҠбҹҒбһҡбһёбһңбҹҒбҹ– $(x^4)' = 4x^3$, $(-4x^2)' = -8x$, $(5x)' = 5$, $(1)' = 0$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $f'(x) = 4x^3 - 8x + 5$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһҠбҹҒбһҡбһёбһңбҹҒбһҸбҹ’бһҡбһ„бҹӢ $x = 1$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = 5x^2 - 7x + 2$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҡбһҖбһҠбҹҒбһҡбһёбһңбҹҒбһ‘бһёбһҳбһҪбһҷбҹ– $f'(x) = 10x - 7$бҹ”
  • бһҮбҹҶбһ“бһҪбһҹ $x = 1$ бһ…бһјбһӣбҹ– $f'(1) = 10(1) - 7 = 3$бҹ”
бһҡбһҖбһҠбҹҒбһҡбһёбһңбҹҒбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \dfrac{12}{x}$ бһӣбһҫбһ…бһ“бҹ’бһӣбҹ„бҹҮ $(0, +\infty)$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҠбҹҒбһҡбһёбһңбҹҒбҹ– $\left(\dfrac{k}{x}\right)' = -\dfrac{k}{x^2}$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $\left(\dfrac{12}{x}\right)' = -\dfrac{12}{x^2}$бҹ”
бһҡбһҖбһ–бҹ’бһҡбһёбһҳбһёбһ‘бһёбһңбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = 8x^7$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • $\int 8x^7 \,dx = 8 \cdot \dfrac{x^8}{8} + C = x^8 + C$бҹ”
бһӮбһҺбһ“бһ¶бһўбһ¶бҹҶбһ„бһҸбҹҒбһҖбҹ’бһҡбһ¶бһӣбһҳбһ·бһ“бһҖбҹҶбһҺбһҸбҹӢ $\int (15x^2 + 6x) \,dx$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • $\int 15x^2 \,dx = 5x^3$ бһ“бһ·бһ„ $\int 6x \,dx = 3x^2$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһҪбһҳбһӮбһә $5x^3 + 3x^2 + C$бҹ”
бһӮбһҺбһ“бһ¶бһўбһ¶бҹҶбһ„бһҸбҹҒбһҖбҹ’бһҡбһ¶бһӣбһҖбҹҶбһҺбһҸбҹӢ $\int_{0}^{5} 5 \,dx$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһ–бҹ’бһҡбһёбһҳбһёбһ‘бһёбһңбһ“бҹғ $5$ бһӮбһә $5x$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $[5x]_0^5 = 5(5) - 5(0) = 25$бҹ”
бһҡбһҖбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $y = \dfrac{2x + 1}{5x - 25}$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ—бһ¶бһӮбһ”бҹӮбһ„бһҒбһ»бһҹбһ–бһёбһҹбһјбһ“бҹ’бһҷбҹ– $5x - 25 \neq 0 \Rightarrow 5x \neq 25 \Rightarrow x \neq 5$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $D = \mathbb{R} \setminus \{5\}$бҹ”
бһҡбһҖбһҹбһҳбһёбһҖбһ¶бһҡбһўбһ¶бһҹбҹҠбһёбһҳбһҸбһјбһҸбһҲбһҡбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $y = \dfrac{3}{x - 12}$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһўбһ¶бһҹбҹҠбһёбһҳбһҸбһјбһҸбһҲбһҡбҹ– бһұбҹ’бһҷбһ—бһ¶бһӮбһ”бҹӮбһ„бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһҹбһјбһ“бҹ’бһҷ $x - 12 = 0 \Rightarrow x = 12$бҹ”
бһҡбһҖбһҹбһҳбһёбһҖбһ¶бһҡбһўбһ¶бһҹбҹҠбһёбһҳбһҸбһјбһҸбһҠбҹҒбһҖбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $y = \dfrac{14x - 3}{7x + 2}$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһўбһ¶бһҹбҹҠбһёбһҳбһҸбһјбһҸбһҠбҹҒбһҖбҹ– $y = \lim_{x \to \pm\infty} \dfrac{14x-3}{7x+2} = \dfrac{14}{7} = 2$бҹ”
бһӮбһҺбһ“бһ¶бһ…бҹҶбһ“бһҪбһ“бһ”бһ“бҹ’бһҹбҹҶ $C_{11}^2$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ”бһ“бҹ’бһҹбҹҶбҹ– $C_{11}^2 = \dfrac{11 \times 10}{2 \times 1} = 55$бҹ”
бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһўбһ”бҹӢбһҳбһҪбһҷбһҳбһ¶бһ“бһҖбһјбһ“бһғбҹ’бһӣбһёбһ–бһҺбҹҢбһҹ бҹ§ бһӮбҹ’бһҡбһ¶бһ”бҹӢ бһ“бһ·бһ„бһ–бһҺбҹҢбһ”бҹғбһҸбһ„ бҹ§ бһӮбҹ’бһҡбһ¶бһ”бҹӢбҹ” бһӮбҹҒбһ…бһ¶бһ”бҹӢбһҷбһҖбһғбҹ’бһӣбһё бҹў бһӮбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹ’бһҡбһҳбһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһ…бҹғбһҠбһ“бҹ’бһҷбҹ” бһҡбһҖбһ…бҹҶбһ“бһҪбһ“бһҖбһҡбһҺбһёбһўбһ¶бһ… $n(S)$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһ…бҹҶбһ“бһҪбһ“бһғбҹ’бһӣбһёбһҹбһҡбһ»бһ” $7 + 7 = 14$бҹ” бһ…бһ¶бһ”бҹӢбһҷбһҖ бҹў бһ–бҹ’бһҡбһҳбһӮбҹ’бһ“бһ¶бҹ– $n(S) = C_{14}^2 = \dfrac{14 \times 13}{2 \times 1} = 91$бҹ”
бһҖбҹ’бһ“бһ»бһ„бһҗбһ„бҹӢбһҳбһҪбһҷбһҳбһ¶бһ“бһ”бһ¶бһӣбҹӢбһ–бһҺбҹҢбһҒбҹҖбһң бҹ§ бһӮбҹ’бһҡбһ¶бһ”бҹӢ бһ“бһ·бһ„бһ–бһҺбҹҢбһӣбһҝбһ„ бҹЈ бһӮбҹ’бһҡбһ¶бһ”бҹӢбҹ” бһӮбҹҒбһ…бһ¶бһ”бҹӢбһҷбһҖбһ”бһ¶бһӣбҹӢ бҹў бһӮбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹ’бһҡбһҳбһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһ…бҹғбһҠбһ“бҹ’бһҷбҹ” бһӮбһҺбһ“бһ¶бһ”бҹ’бһҡбһјбһ”бһ¶бһ”бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ¶бһ“ В«бһ”бһ¶бһӣбҹӢбһ–бһҺбҹҢбһҒбҹҖбһңбһ‘бһ¶бҹҶбһ„ бҹў бһӮбҹ’бһҡбһ¶бһ”бҹӢВ»бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҖбһҡбһҺбһёбһўбһ¶бһ…бҹ– $n(S) = C_{10}^2 = 45$бҹ”
  • бһҖбһҡбһҺбһёбһҹбҹ’бһҡбһ” (бһҒбҹҖбһң бҹў бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳ бҹ§)бҹ– $n(A) = C_7^2 = 21$бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бҹ’бһҡбһјбһ”бһ¶бһ”бҹ– $P(A) = \dfrac{21}{45} = \dfrac{7}{15}$бҹ”
бһӮбҹҒбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҹбһ·бһҹбҹ’бһҹ бҹў бһ“бһ¶бһҖбҹӢбһ–бҹ’бһҡбһҳбһӮбҹ’бһ“бһ¶ бһ…бҹҒбһүбһ–бһёбһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ·бһҹбҹ’бһҹбһ”бҹ’бһҡбһ»бһҹ бҹЈ бһ“бһ¶бһҖбҹӢ бһ“бһ·бһ„бһҹбҹ’бһҡбһё бҹ§ бһ“бһ¶бһҖбҹӢбҹ” бһҡбһҖбһ…бҹҶбһ“бһҪбһ“бһҖбһҡбһҺбһёбһҹбҹ’бһҡбһ”бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ¶бһ“ В«бһҹбһ·бһҹбҹ’бһҹбһ”бҹ’бһҡбһ»бһҹбһ‘бһ¶бҹҶбһ„ бҹў бһ“бһ¶бһҖбҹӢВ»бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҡбһҫбһҹбһҹбһ·бһҹбҹ’бһҹбһ”бҹ’бһҡбһ»бһҹ бҹў бһ“бһ¶бһҖбҹӢ бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳбһ”бҹ’бһҡбһ»бһҹбһ‘бһ¶бҹҶбһ„ бҹЈ бһ“бһ¶бһҖбҹӢбҹ– $n(A) = C_3^2 = 3$бҹ”
бһҡбһҖбһҖбһјбһўбһҡбһҠбҹ„бһ“бҹҒбһҖбҹҶбһ–бһјбһӣ $V$ бһ“бҹғбһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһёбһҖбһ¶бһҡ $(y - 5)^2 = 8(x + 1)$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡбһҳбһ¶бһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ $(y - k)^2 = 4p(x - h)$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҖбҹҶбһ–бһјбһӣ $V(h, k) = V(-1, 5)$бҹ”
бһҡбһҖбһҸбһҳбҹ’бһӣбҹғ $p$ бһ“бҹғбһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһёбһҖбһ¶бһҡ $x^2 = -32y$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбһҳбһёбһҖбһ¶бһҡбһӮбһә $x^2 = 4py$бҹ” бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҳбҹҒбһӮбһ»бһҺбҹ– $4p = -32 \Rightarrow p = \dfrac{-32}{4} = -8$бҹ”
бһҡбһҖбһҹбһҳбһёбһҖбһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһҹ (Directrix) бһ“бҹғбһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣ $y^2 = -16x$ (бһҖбҹҶбһ–бһјбһӣбһ“бҹ… $V(0,0)$)бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡ $y^2 = 4px$ бһҳбһ¶бһ“ $4p = -16 \Rightarrow p = -4$бҹ”
  • бһҹбһҳбһёбһҖбһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹҒбҹҮбһӮбһә $x = -p \Rightarrow x = -(-4) = 4$бҹ”
бһ”бһҫбһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣбһҳбһ¶бһ“бһҹбһҳбһёбһҖбһ¶бһҡ $(x - 7)^2 = 8(y + 2)$ бһҸбһҫбһўбҹҗбһҖбҹ’бһҹбһҶбҹ’бһӣбһ»бҹҮбһҡбһ”бһҹбҹӢбһңбһ¶бһҮбһ¶бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҺбһ¶? (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡбһҳбһ¶бһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢ $(x - h)^2 = 4p(y - k)$ бһӮбһәбһҮбһ¶бһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһўбҹҗбһҖбҹ’бһҹбһҶбҹ’бһӣбһ»бҹҮбһҮбһ¶бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҲбһҡ $x = h$ бһӮбһә $x = 7$бҹ”
бһҡбһҖбһҖбһјбһўбһҡбһҠбҹ„бһ“бҹҒбһ•бҹ’бһ…бһ·бһҸ $I$ бһ“бҹғбһўбҹҒбһӣбһёбһ”бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһёбһҖбһ¶бһҡ $\dfrac{(x - 6)^2}{49} + \dfrac{(y + 7)^2}{81} = 1$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһ•бҹ’бһўбҹӮбһҖбһҸбһ¶бһҳбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ $\dfrac{(x - h)^2}{a^2} + \dfrac{(y - k)^2}{b^2} = 1$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ•бҹ’бһ…бһ·бһҸ $I(h, k) = I(6, -7)$бҹ”
бһ–бһёбһҹбһҳбһёбһҖбһ¶бһҡбһўбҹҒбһӣбһёбһ” $\dfrac{x^2}{25} + \dfrac{y^2}{121} = 1$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ”бҹ’бһҡбһңбҹӮбһ„бһ–бһ¶бһҖбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣбһўбҹҗбһҖбҹ’бһҹбһ’бҹҶ ($a$)бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҸбһҳбҹ’бһӣбҹғбһ’бҹҶбһҮбһ¶бһ„бһӮбҹҒбһӮбһәбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳ $y^2$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $a^2 = 121 \Rightarrow a = \sqrt{121} = 11$бҹ”
бһ–бһёбһҹбһҳбһёбһҖбһ¶бһҡбһўбҹҒбһӣбһёбһ”бһҠбһҠбҹӮбһӣ $\dfrac{x^2}{25} + \dfrac{y^2}{121} = 1$ бһҸбһҫбһўбҹҗбһҖбҹ’бһҹбһ’бҹҶбһҡбһ”бһҹбҹӢбһңбһ¶бһҮбһ¶бһўбҹҗбһҖбҹ’бһҹбһҠбҹҒбһҖбһ¬бһўбҹҗбһҖбҹ’бһҹбһҲбһҡ? (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҠбҹ„бһҷбһҹбһ¶бһҡ $a^2 = 121$ бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳбһҮбҹҶбһ–бһјбһҖ $y^2$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһўбҹҒбһӣбһёбһ”бһ“бҹҒбҹҮбһҳбһ¶бһ“бһўбҹҗбһҖбҹ’бһҹбһ’бҹҶбһҮбһ¶бһўбҹҗбһҖбҹ’бһҹбһҲбһҡбһ“бҹ…бһӣбһҫбһўбҹҗбһҖбҹ’бһҹбһўбһҡбһҠбҹ„бһ“бҹҒ ($y'Oy$)бҹ”
бһҡбһҖбһ”бҹ’бһҡбһңбҹӮбһ„бһўбҹҗбһҖбҹ’бһҹбһ’бҹҶ бһ“бҹғбһўбҹҒбһӣбһёбһ”бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһёбһҖбһ¶бһҡ $\dfrac{x^2}{49} + \dfrac{y^2}{16} = 1$бҹ” (бҹӨ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҸбһҳбҹ’бһӣбҹғбһ’бҹҶбһҮбһ¶бһ„бһӮбһә $a^2 = 49 \Rightarrow a = 7$ (бһ–бһ¶бһҖбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣбһўбҹҗбһҖбҹ’бһҹбһ’бҹҶ)бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бҹ’бһҡбһңбҹӮбһ„бһўбҹҗбһҖбҹ’бһҹбһ’бҹҶбһҹбһҡбһ»бһ”бһӮбһә $2a = 2(7) = 14$бҹ”
