рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бһӮбһҺбһ“бһ¶бһӣбһёбһҳбһёбһҸ $\lim_{x \to 1} (7x - 3)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҮбҹҶбһ“бһҪбһҹ $x = 1$ бһ…бһјбһӣбҹ– $\lim_{x \to 1} (7x - 3) = 7(1) - 3 = 4$бҹ”
бһӮбһҺбһ“бһ¶бһӣбһёбһҳбһёбһҸ $\lim_{x \to +\infty} \dfrac{12x^2 + 1}{4x^2 - 3}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҡбһ¶бһ„бһҳбһ·бһ“бһҖбҹҶбһҺбһҸбҹӢ $\dfrac{\infty}{\infty}$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҠбһәбһҖбҹ’бһҡбҹҒбһҹбҹ’бһҳбһҫбһӮбҹ’бһ“бһ¶бҹ– бһҷбһҖбһҳбҹҒбһӮбһ»бһҺбһҠбһәбһҖбҹ’бһҡбҹҒбһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбһ…бҹӮбһҖбһӮбҹ’бһ“бһ¶ бһӮбһә $\dfrac{12}{4} = 3$бҹ”
бһӮбһҺбһ“бһ¶бһӣбһёбһҳбһёбһҸ $\lim_{x \to 5} \dfrac{x^2 - 25}{x - 5}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҡбһ¶бһ„бһҳбһ·бһ“бһҖбҹҶбһҺбһҸбҹӢ $\dfrac{0}{0}$бҹ– бһ”бҹҶбһ”бҹӮбһҖбһ—бһ¶бһӮбһҷбһҖ $(x-5)(x+5)$ бһҡбһҪбһ…бһҹбһҳбҹ’бһҡбһҪбһӣбһ“бһ№бһ„бһ—бһ¶бһӮбһ”бҹӮбһ„ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $\lim_{x \to 5} (x + 5) = 5 + 5 = 10$бҹ”
бһӮбһҺбһ“бһ¶бһӣбһёбһҳбһёбһҸ $\lim_{x \to 4} \dfrac{x^2 - 5x + 4}{x - 4}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһ”бҹҶбһ”бҹӮбһҖбһ—бһ¶бһӮбһҷбһҖбҹ– $x^2 - 5x + 4 = (x-4)(x-1)$бҹ”
  • бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһҸбҹ’бһҸбһ¶бһҡбһҪбһҳ $(x-4)$ бһ…бҹҒбһү бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $\lim_{x \to 4} (x - 1) = 4 - 1 = 3$бҹ”
бһҡбһҖбһҠбҹҒбһҡбһёбһңбҹҒбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = 3x^4 - 5x^3 + 2$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҠбҹҒбһҡбһёбһңбҹҒбҹ– $(3x^4)' = 12x^3$, $(-5x^3)' = -15x^2$, $(2)' = 0$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $f'(x) = 12x^3 - 15x^2$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһҠбҹҒбһҡбһёбһңбҹҒбһҸбҹ’бһҡбһ„бҹӢ $x = 0$ бһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = 5x^3 - 2x + 9$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҡбһҖбһҠбҹҒбһҡбһёбһңбҹҒбһ‘бһёбһҳбһҪбһҷбҹ– $f'(x) = 15x^2 - 2$бҹ”
  • бһҮбҹҶбһ“бһҪбһҹ $x = 0$ бһ…бһјбһӣбҹ– $f'(0) = 15(0)^2 - 2 = -2$бҹ”
бһҡбһҖбһҠбҹҒбһҡбһёбһңбҹҒбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = \dfrac{1}{2x}$ бһӣбһҫбһ…бһ“бҹ’бһӣбҹ„бҹҮ $(0, +\infty)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡбһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡ $f(x) = \dfrac{1}{2} \cdot \dfrac{1}{x}$бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $f'(x) = \dfrac{1}{2} \cdot \left(-\dfrac{1}{x^2}\right) = -\dfrac{1}{2x^2}$бҹ”
бһҡбһҖбһ–бҹ’бһҡбһёбһҳбһёбһ‘бһёбһңбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $f(x) = 7x^6$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • $\int 7x^6 \,dx = 7 \cdot \dfrac{x^7}{7} + C = x^7 + C$бҹ”
бһӮбһҺбһ“бһ¶бһўбһ¶бҹҶбһ„бһҸбҹҒбһҖбҹ’бһҡбһ¶бһӣбһҳбһ·бһ“бһҖбҹҶбһҺбһҸбҹӢ $\int (9x^2 + 2x - 5) \,dx$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • $\int 9x^2 \,dx = 3x^3$, $\int 2x \,dx = x^2$ бһ“бһ·бһ„ $\int (-5) \,dx = -5x$бҹ” бһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһҪбһҳбһӮбһә $3x^3 + x^2 - 5x + C$бҹ”
бһӮбһҺбһ“бһ¶бһўбһ¶бҹҶбһ„бһҸбҹҒбһҖбҹ’бһҡбһ¶бһӣбһҖбҹҶбһҺбһҸбҹӢ $\int_{0}^{2} 3x^2 \,dx$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһ–бҹ’бһҡбһёбһҳбһёбһ‘бһёбһңбһ“бҹғ $3x^2$ бһӮбһә $x^3$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $[x^3]_0^2 = 2^3 - 0^3 = 8$бҹ”
бһҡбһҖбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $y = \dfrac{7x}{2x - 10}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ—бһ¶бһӮбһ”бҹӮбһ„бһҒбһ»бһҹбһ–бһёбһҹбһјбһ“бҹ’бһҷбҹ– $2x - 10 \neq 0 \Rightarrow 2x \neq 10 \Rightarrow x \neq 5$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $D = \mathbb{R} \setminus \{5\}$бҹ”
бһҡбһҖбһҹбһҳбһёбһҖбһ¶бһҡбһўбһ¶бһҹбҹҠбһёбһҳбһҸбһјбһҸбһҲбһҡбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $y = \dfrac{x - 3}{x + 10}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһұбҹ’бһҷбһ—бһ¶бһӮбһ”бҹӮбһ„бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһҹбһјбһ“бҹ’бһҷбҹ– $x + 10 = 0 \Rightarrow x = -10$бҹ”
бһҡбһҖбһҹбһҳбһёбһҖбһ¶бһҡбһўбһ¶бһҹбҹҠбһёбһҳбһҸбһјбһҸбһҠбҹҒбһҖбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ $y = \dfrac{10x + 5}{5x - 2}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһўбһ¶бһҹбҹҠбһёбһҳбһҸбһјбһҸбһҠбҹҒбһҖбҹ– $y = \lim_{x \to \pm\infty} \dfrac{10x+5}{5x-2} = \dfrac{10}{5} = 2$бҹ”
бһӮбһҺбһ“бһ¶бһ…бҹҶбһ“бһҪбһ“бһ”бһ“бҹ’бһҹбҹҶ $C_9^2$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ”бһ“бҹ’бһҹбҹҶбҹ– $C_9^2 = \dfrac{9 \times 8}{2 \times 1} = 36$бҹ”
бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһўбһ”бҹӢбһҳбһҪбһҷбһҳбһ¶бһ“бһ”бҹҠбһ·бһ…бһ–бһҺбҹҢбһҒбҹҖбһң бҹҘ бһҠбһҫбһҳ бһ“бһ·бһ„бһ–бһҺбҹҢбһҖбҹ’бһҡбһ бһҳ бҹҘ бһҠбһҫбһҳбҹ” бһӮбҹҒбһҠбһҖбһҷбһҖбһ”бҹҠбһ·бһ… бҹў бһҠбһҫбһҳбһ–бҹ’бһҡбһҳбһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһ…бҹғбһҠбһ“бҹ’бһҷбҹ” бһҡбһҖбһ…бҹҶбһ“бһҪбһ“бһҖбһҡбһҺбһёбһўбһ¶бһ… $n(S)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһ…бҹҶбһ“бһҪбһ“бһ”бҹҠбһ·бһ…бһҹбһҡбһ»бһ” $5 + 5 = 10$бҹ” бһ…бһ¶бһ”бҹӢбһҷбһҖ бҹў бһ–бҹ’бһҡбһҳбһӮбҹ’бһ“бһ¶бҹ– $n(S) = C_{10}^2 = \dfrac{10 \times 9}{2 \times 1} = 45$бҹ”
бһҖбҹ’бһ“бһ»бһ„бһҗбһ„бҹӢбһҳбһҪбһҷбһҳбһ¶бһ“бһ•бҹ’бһӣбҹӮбһҖбҹ’бһҡбһјбһ… бҹӨ бһ•бҹ’бһӣбҹӮ бһ“бһ·бһ„бһ•бҹ’бһӣбҹӮбһ”бҹүбҹ„бһҳ бҹҘ бһ•бҹ’бһӣбҹӮбҹ” бһӮбҹҒбһӣбһјбһҖбһҷбһҖбһ•бҹ’бһӣбҹӮбһҲбһҫ бҹў бһ•бҹ’бһӣбҹӮбһ–бҹ’бһҡбһҳбһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһ…бҹғбһҠбһ“бҹ’бһҷбҹ” бһӮбһҺбһ“бһ¶бһ”бҹ’бһҡбһјбһ”бһ¶бһ”бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ¶бһ“ В«бһ•бҹ’бһӣбҹӮбһҖбҹ’бһҡбһјбһ…бһ‘бһ¶бҹҶбһ„ бҹўВ»бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҖбһҡбһҺбһёбһўбһ¶бһ…бҹ– $n(S) = C_9^2 = 36$бҹ”
  • бһҖбһҡбһҺбһёбһҹбҹ’бһҡбһ” (бһҖбҹ’бһҡбһјбһ… бҹў бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳ бҹӨ)бҹ– $n(A) = C_4^2 = 6$бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бҹ’бһҡбһјбһ”бһ¶бһ”бҹ– $P(A) = \dfrac{6}{36} = \dfrac{1}{6}$бҹ”
бһӮбһҺбҹҲбһҖбһҳбҹ’бһҳбһҖбһ¶бһҡбһҳбһҪбһҷбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһҫбһҹбһҹбһҳбһ¶бһҮбһ·бһҖ бҹў бһ“бһ¶бһҖбҹӢ бһ…бҹҒбһүбһ–бһёбһ”бһ»бһҡбһҹ бҹӨ бһ“бһ¶бһҖбҹӢ бһ“бһ·бһ„бһҹбҹ’бһҸбҹ’бһҡбһё бҹӨ бһ“бһ¶бһҖбҹӢбҹ” бһҡбһҖбһ…бҹҶбһ“бһҪбһ“бһҖбһҡбһҺбһёбһҹбҹ’бһҡбһ”бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ¶бһ“ В«бһ”бһ»бһҡбһҹбһ‘бһ¶бҹҶбһ„ бҹў бһ“бһ¶бһҖбҹӢВ»бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҡбһҫбһҹбһ”бһ»бһҡбһҹ бҹў бһ“бһ¶бһҖбҹӢ бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳбһ”бһ»бһҡбһҹбһ‘бһ¶бҹҶбһ„ бҹӨ бһ“бһ¶бһҖбҹӢбҹ– $n(A) = C_4^2 = 6$бҹ”
бһҡбһҖбһҖбһјбһўбһҡбһҠбҹ„бһ“бҹҒбһҖбҹҶбһ–бһјбһӣ $V$ бһ“бҹғбһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһёбһҖбһ¶бһҡ $(y + 1)^2 = 4(x - 3)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡбһҳбһ¶бһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ $(y - k)^2 = 4p(x - h)$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҖбҹҶбһ–бһјбһӣ $V(h, k) = V(3, -1)$бҹ”
бһҡбһҖбһҸбһҳбҹ’бһӣбҹғ $p$ бһ“бҹғбһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһёбһҖбһ¶бһҡ $y^2 = 32x$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбһҳбһёбһҖбһ¶бһҡбһӮбһә $y^2 = 4px$бҹ” бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҳбҹҒбһӮбһ»бһҺбҹ– $4p = 32 \Rightarrow p = \dfrac{32}{4} = 8$бҹ”
бһҡбһҖбһҹбһҳбһёбһҖбһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһҹ (Directrix) бһ“бҹғбһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣ $x^2 = -4y$ (бһҖбҹҶбһ–бһјбһӣбһ“бҹ… $V(0,0)$)бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡ $x^2 = 4py$ бһҳбһ¶бһ“ $4p = -4 \Rightarrow p = -1$ (бһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣбһ”бҹӮбһҡбһ…бһ»бҹҮбһҖбҹ’бһҡбҹ„бһҳ)бҹ”
  • бһҹбһҳбһёбһҖбһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһҹбһӮбһә $y = -p \Rightarrow y = -(-1) = 1$бҹ”
бһ”бһҫбһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣбһҳбһ¶бһ“бһҹбһҳбһёбһҖбһ¶бһҡ $(y - 6)^2 = 16(x + 2)$ бһҸбһҫбһўбҹҗбһҖбҹ’бһҹбһҶбҹ’бһӣбһ»бҹҮбһҡбһ”бһҹбҹӢбһңбһ¶бһҮбһ¶бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҺбһ¶? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡбһҳбһ¶бһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢ $(y - k)^2 = 4p(x - h)$ бһӮбһәбһҮбһ¶бһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһўбҹҗбһҖбҹ’бһҹбһҶбҹ’бһӣбһ»бҹҮбһҮбһ¶бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҠбҹҒбһҖ $y = k$ бһӮбһә $y = 6$бҹ”
бһҡбһҖбһҖбһјбһўбһҡбһҠбҹ„бһ“бҹҒбһ•бҹ’бһ…бһ·бһҸ $I$ бһ“бҹғбһўбҹҒбһӣбһёбһ”бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһёбһҖбһ¶бһҡ $\dfrac{(x - 4)^2}{49} + \dfrac{(y + 5)^2}{36} = 1$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһ•бҹ’бһўбҹӮбһҖбһҸбһ¶бһҳбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ $\dfrac{(x - h)^2}{a^2} + \dfrac{(y - k)^2}{b^2} = 1$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ•бҹ’бһ…бһ·бһҸ $I(h, k) = I(4, -5)$бҹ”
бһ–бһёбһҹбһҳбһёбһҖбһ¶бһҡбһўбҹҒбһӣбһёбһ” $\dfrac{x^2}{4} + \dfrac{y^2}{81} = 1$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ”бҹ’бһҡбһңбҹӮбһ„бһ–бһ¶бһҖбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣбһўбҹҗбһҖбҹ’бһҹбһ’бҹҶ ($a$)бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҸбһҳбҹ’бһӣбҹғбһ’бҹҶбһҮбһ¶бһ„бһӮбҹҒбһӮбһәбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳ $y^2$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $a^2 = 81 \Rightarrow a = \sqrt{81} = 9$бҹ”
бһ–бһёбһҹбһҳбһёбһҖбһ¶бһҡбһўбҹҒбһӣбһёбһ”бһҠбһҠбҹӮбһӣ $\dfrac{x^2}{4} + \dfrac{y^2}{81} = 1$ бһҸбһҫбһўбҹҗбһҖбҹ’бһҹбһ’бҹҶбһҡбһ”бһҹбҹӢбһңбһ¶бһҮбһ¶бһўбҹҗбһҖбҹ’бһҹбһҠбҹҒбһҖбһ¬бһҲбһҡ? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҠбҹ„бһҷбһҹбһ¶бһҡ $a^2 = 81$ бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳбһҮбҹҶбһ–бһјбһҖ $y^2$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһўбҹҒбһӣбһёбһ”бһ“бҹҒбҹҮбһҳбһ¶бһ“бһўбҹҗбһҖбҹ’бһҹбһ’бҹҶбһҲбһҡбһ“бҹ…бһӣбһҫбһўбҹҗбһҖбҹ’бһҹбһўбһҡбһҠбҹ„бһ“бҹҒ ($y'Oy$)бҹ”
бһҡбһҖбһ”бҹ’бһҡбһңбҹӮбһ„бһўбҹҗбһҖбҹ’бһҹбһҸбһјбһ… бһ“бҹғбһўбҹҒбһӣбһёбһ”бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһёбһҖбһ¶бһҡ $\dfrac{x^2}{16} + \dfrac{y^2}{121} = 1$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
  • бһҸбһҳбҹ’бһӣбҹғбһҸбһјбһ…бһҮбһ¶бһ„бһӮбһә $b^2 = 16 \Rightarrow b = 4$ (бһ–бһ¶бһҖбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣбһўбҹҗбһҖбҹ’бһҹбһҸбһјбһ…)бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бҹ’бһҡбһңбҹӮбһ„бһўбҹҗбһҖбҹ’бһҹбһҸбһјбһ…бһҹбһҡбһ»бһ”бһӮбһә $2b = 2(4) = 8$бҹ”
